# DriveNow Marketplace (Demo)

A modern, single-page used-car marketplace demo built with plain HTML, CSS, and JavaScript.

## Run locally in Visual Studio Code

1. Open this folder in VS Code.
2. Install the **Live Server** extension (recommended), then right-click `index.html` and choose **Open with Live Server**.

### Alternative (no extension)

Run a local static server:

```bash
python3 -m http.server 3000
```

Then open:

- `http://localhost:3000`

## Features

- Modern homepage hero with search filters
- Inventory browsing with save/favorite support
- Vehicle detail pages with image gallery and specs
- Test checkout flow (always succeeds)
- Purchase confirmation page with animated success indicator
- Dark/light theme toggle
- Responsive design for mobile, tablet, and desktop
- Live chat demo widget

## Notes

- This is a **demo/test-only** checkout experience.
- No real payment processing is used.
